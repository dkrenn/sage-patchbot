The SageMath patchbot relies on volunteers to contribute CPU.

If you would like to see your patches get tested quicker, on architectures you care about, consider contributing yourself. Install with

sage -i http://sage.math.washington.edu/home/robertwb/patches/patchbot-2.3.4.spkg
or sage -i http://chapoton.perso.math.cnrs.fr/patchbot-2.3.4.spkg
and run with 
sage -patchbot
in an unused SageMath install. The patchbot supports various configurations, e.g. to give preference to components you care about or to run only at night. See the wiki for more details.
