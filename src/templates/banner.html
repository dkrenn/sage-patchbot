The Sage patchbot relies on volunteers to contribute CPU.

If you would like to see your patches get tested quicker, on architectures you care about, consider contributing yourself by running 

sage -i http://sage.math.washington.edu/home/robertwb/patches/patchbot-2.3.1.spkg
(or sage -i http://chapoton.perso.math.cnrs.fr/patchbot-2.3.1.spkg)
sage -patchbot
in an unused Sage install. The patchbot supports various configurations, e.g. to give preference to components you care about or to run only at night. See the wiki for more details.
